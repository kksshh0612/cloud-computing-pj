# 클라우드 컴퓨팅 

## AWS EC2 동적 자원 관리 프로그램

![image](https://github.com/user-attachments/assets/0675fee0-4425-4b6c-9d9a-2e9c38108a5c)

### 기능
> 1. 모든 인스턴스 정보를 출력
> 2. 가용 영역을 출력
> 3. 특정 인스턴스 시작
> 4. 가용 리젼 출력
> 5. 특정 인스턴스 중지
> 6. AMI로 인스턴스 생성
> 7. 특정 인스턴스 재부팅
> 8. 모든 AMI 출력
> 9. condor status 출력
> 10. 모든 인스턴스 CPU 사용량 (1시간 평균), Network 인바운드/아웃바운드 트래픽 데이터 양(10분 평균) 출력
> 99. 종료
